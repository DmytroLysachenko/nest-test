# UI Component Library Overview

This component library is built with React for admin and dashboard scenarios, with theme customization and a modern developer experience. The style is inspired by [shadcn/ui](https://ui.shadcn.com/) and tailored to team needs.

---

## Quick Start

**Install dependencies**

```bash
pnpm add @repo/ui
```

**Import global styles**

In your entry file (for example `main.tsx` or `app.tsx`):

```ts
import '@repo/ui/styles/globals.css';
```

**Use components**

```tsx
import { Button } from '@repo/ui/components/button';

<Button variant="primary">Button</Button>
```

---

## Components

- **Button**: multiple variants and sizes, supports icons and loading state
- **Avatar**: user avatar display
- **DropdownMenu**: multi-level menus and interactions
- **Sonner**: lightweight notifications
- **Label**, **Input**, **Form**, **Card**, and more

See component docs in the left sidebar.

---

## Theme and customization

- Theme colors and dark mode: see [Colors](/?path=/docs/colors)
- Customize CSS variables in `globals.css`
- Extend with Tailwind CSS utilities as needed
