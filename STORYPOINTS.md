# Story Points

This file tracks milestones for the career search assistant build. Keep it short and update after each significant feature.

## Milestone 1: Core Intake + AI (Done)
- Auth and user management
- Profile inputs (target roles + notes)
- Document uploads via GCS (signed URL + confirm)
- Career profile generation (Gemini)

## Milestone 2: Extraction + Matching (Next)
- PDF text extraction pipeline
- Structured profile refinement
- Job matching prompt + scoring

## Milestone 3: Job Tracker (Planned)
- Application notebook (company, link, status, notes)
- Reminders and follow-ups

## Milestone 4: Automation (Planned)
- Scheduled crawlers
- Async AI jobs (queue, retries, monitoring)
